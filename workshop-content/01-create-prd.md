# Create a PRD (Project Requirements Document)

Prompt (same for VS Code and CLI):

```
/plan I want to create a simple kanban app. Help me make a complete PRD for it. Ask me any clarifying questions as necessary using #askQuestions. Here are the requirements:

- It should have 3 columns: todo, in progress, and complete
- You can add new cards to the board with an "+ Add card" button
- A card consists of a title and a body
- You can move cards between columns with a dropdown in each card
- No backend, just use localStorage to store data
```
