# Create a Drag-and-Drop Spec

Prompt (same for VS Code and CLI):

```text
Help me create a specification for a drag-and-drop feature for the cards. Split the specification out into sections so I can implement and review them one at a time.

Each section should be small enough such that I can review the work done in 5-10 minutes. Each of the sections should contain a set of specific implementation tasks to complete the technical requirements. Use checkboxes for each one.

The overall spec should contain a user story (as an x, I want to do y), technical requirements, acceptance criteria, definition of done, and any other sections you think might be helpful.

Save this spec as a new markdown file. Ask me any clarifying questions with #askQuestions.
```
